# Flappy Puppy

A Flappy Bird-style game built in C using SDL2.

## Status
🚧 Work in progress

## What's Implemented
- SDL2 window and renderer setup
- Player with gravity and flap impulse (spacebar)
- Ground collision
- Delta time game loop

## Planned Features
- Pipe spawning and movement
- Collision detection
- Scoring system
- Game over screen

## Dependencies
- C compiler (gcc or clang)
- SDL2

## Build & Run
```bash
gcc main.c -o Flappy_Bird -lSDL2
./Flappy_Bird
```
